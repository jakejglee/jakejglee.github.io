bellsprout - universal garden planner

Sustainable farming practices (permaculture) are challenging to implement on a commercial level largely due to harvesting challenges; namely identifying ripe produce, appropriate handling of produce, and making many complex design decisions. Project bellsprout aims to help tackle the first and third of these issues by providing design recommendations via a web-app. By combining as much data as can be made available (user input and sensor data) into a three-dimensional grid containing descriptive data at each coordinate, and a database of planting guidelines, bellsprout will be able to provide a farmer (or gardener) with useful information before they plant crops.
